# MyMakeup - Landing Page Responsive

Bienvenida/o a mi proyecto final del módulo Fundamentos de desarrollo web.  
En este proyecto desarrollé y desplegué una Landing Page responsiva de una tienda de productos de maquillaje con fines de demostración.

## Instalación

1. Clona este repositorio:

   ```
   git clone https://github.com/antodollparts/responsive-page.git
   ```

2. Navega al directorio del proyecto:

   ```
   cd responsive-page
   ```

3. En Visual Studio Code descargar la extensión para visualizar la página:

   ```
   Live Server
   ```

4. Ejecuta la extensión:

   ```
   Visualiza el proyecto en tu navegador web
   ```
## Tecnologías utilizadas

- HTML5
- CSS3
- JS
